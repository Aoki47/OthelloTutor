# 年末調整PDF解析アプリ 仕様書

## 1. 概要

### 1.1 アプリ名（仮）
**NeNcho Check（年調チェック）**

### 1.2 目的
源泉徴収票・給料明細・確定申告書のPDFをブラウザに読み込み、以下を自動で確認・解説する：

- 所得税・住民税・社会保険料の計算が妥当か
- ふるさと納税の控除が正しく反映されているか
- 見落としがちな控除が適用されているか

### 1.3 基本方針
- **完全ブラウザ動作**：PDFを含むすべてのデータをサーバーに送信しない
- **個人利用向け**：シンプルなUI、詳細な解説テキスト付き
- **2024年度（令和6年分）の税制に対応**

## 3. 技術仕様

### 3.1 技術スタック

| 分類 | 採用技術 | 理由 |
|------|----------|------|
| フレームワーク | React + TypeScript | 複数PDFの状態管理・型安全性 |
| PDF解析 | [PDF.js](https://mozilla.github.io/pdf.js/)（Mozilla製） | ブラウザ完結、実績あり |
| スタイル | TailwindCSS | 素早いUI構築 |
| ビルド | Vite | 高速・設定シンプル |
| ホスティング | GitHub Pages / Vercel（静的ファイルのみ） | 無料・簡単 |

### 3.2 動作環境
- モダンブラウザ（Chrome 90+, Firefox 90+, Safari 15+）
- スマートフォンは対象外（PCブラウザのみ想定）

### 3.3 データの扱い
- アップロードされたPDFはブラウザのメモリ内のみで処理
- localStorageには補足入力値のみ任意保存（デフォルトOFF）
- 外部APIへの通信なし

4. 抽出に失敗した項目はUIで明示し、手入力を促す
5. 電子データ版（テキスト埋め込み）が対象。スキャンPDFは非対応（明記）

## 7. 計算・チェックロジック

### 7.1 所得税の計算検証（2024年分）

```
1. 給与所得控除の計算（速算表）
   年収 162.5万円以下      → 55万円
   年収 162.5〜180万円     → 年収 × 40% - 10万円
   年収 180〜360万円       → 年収 × 30% + 8万円
   年収 360〜660万円       → 年収 × 20% + 44万円
   年収 660〜850万円       → 年収 × 10% + 110万円
   年収 850万円超          → 195万円（上限）
   給与所得 = 年収 - 給与所得控除額

2. 所得控除の積み上げ
   基礎控除：48万円（合計所得2,400万円以下）
   社会保険料控除：PDF抽出値
   生命保険料控除：PDF抽出値（上限12万円）
   地震保険料控除：PDF抽出値（上限5万円）
   配偶者控除 / 配偶者特別控除（補足入力より）
   扶養控除（補足入力より。16歳未満は対象外）
   特定扶養控除（19〜22歳：63万円）
   老人扶養控除（70歳以上：48〜58万円）
   iDeCo控除（補足入力より）
   寄附金控除（ふるさと納税・確定申告の場合のみ）
   医療費控除（補足入力より・確定申告が必要）

3. 課税所得 = 給与所得 - 所得控除合計（1,000円未満切捨て）

4. 所得税額（超過累進税率）
   195万円以下       → 5%
   195〜330万円      → 10%（控除 97,500円）
   330〜695万円      → 20%（控除 427,500円）
   695〜900万円      → 23%（控除 636,000円）
   900〜1,800万円    → 33%（控除 1,536,000円）
   1,800〜4,000万円  → 40%（控除 2,796,000円）
   4,000万円超       → 45%（控除 4,796,000円）

5. 復興特別所得税 = 所得税額 × 2.1%

6. 税額控除の適用
   住宅借入金等特別控除を差し引く（税額控除）

7. 検証
   計算値 vs PDF記載の源泉徴収税額
   差異 ±5,000円以内  → ✅ 正常
   差異 ±5,001〜50,000円 → ⚠️ 要確認
   差異 ±50,001円超   → ❌ 異常の可能性
```

### 7.2 住民税の確認（給料明細からの実額取得）

住民税は**給料明細の特別徴収欄から実額を直接読み取る**。推定計算は明細が不足している場合の補助として使用する。

```
【メインフロー：給料明細が読み込まれている場合】

1. 月次住民税の集計
   各月の給料明細から「住民税（特別徴収）」欄の金額を抽出
   年間住民税合計 = 各月の住民税額の合計

2. 住民税の適用期間の確認
   住民税は前年所得に基づき当年6月〜翌年5月に天引きされる
   → 1〜5月分：前々年所得ベースの住民税
   → 6〜12月分：前年所得ベースの住民税
   この切り替わりをUIで明示する

3. 前年所得ベースの住民税（6〜12月の7ヶ月分）を表示
   月額 × 12換算で年間見込みも表示

4. 住民税の妥当性チェック
   6月以降の月額が一定かどうか確認（特殊事情がなければ均等額）
   月ごとにばらつきがある場合 → ⚠️ 特別徴収の変更・引越し等の可能性を案内

5. 推定値との比較（検証用）
   以下の簡易計算で推定値を算出し、実額と照合
   
   課税所得（住民税用）= 給与所得 - 住民税所得控除合計
   　基礎控除：43万円
   　社会保険料控除・扶養控除等：源泉徴収票の値を使用
   
   住民税所得割 = 課税所得 × 10%
   調整控除を差し引く
   均等割（5,000円）を加算
   
   推定値 vs 給料明細集計値
   ±15%以内 → ✅ 概ね一致（前年所得との差・控除の差で生じる誤差）
   ±15%超  → ⚠️ 差異の原因を解説（前年と今年で収入や控除が大きく変わった等）

【フォールバック：給料明細が未読み込みの場合】

簡易推定値のみ表示：
   「給料明細を読み込むと、実際の住民税額を確認できます」
   として案内し、推定額を参考値として表示する
```

### 7.3 社会保険料の妥当性チェック（都道府県別）

```
健康保険料（協会けんぽ・都道府県別）
   標準報酬月額 = 年収 ÷ 12 → 等級表で決定（上限139万円、下限5.8万円）
   健康保険料（年）= 標準報酬月額 × 都道府県別料率（本人負担分 = 料率 ÷ 2）× 12
   ※ 40歳以上は介護保険料率（全国一律 1.60%）を加算

厚生年金保険料
   標準報酬月額（上限65万円）× 9.15% × 12

雇用保険料
   年収 × 0.6%（一般の事業）

社会保険料合計の比較
   推定値 vs PDF記載の社会保険料等の金額
   ±10%以内 → ✅ 都道府県・等級を考慮した結果として妥当
   ±10%超  → ⚠️ 組合健保・標準報酬月額の変動月等で差が出る場合を説明

都道府県別料率マスタ（2024年度・協会けんぽ）
   北海道: 10.21% / 青森: 9.70% / 岩手: 9.76% / …（全47都道府県収録）
```

### 7.4 ふるさと納税チェック

```
1. ふるさと納税上限額の計算
   住民税所得割（7.2で計算した値）を使用
   上限の目安 = 住民税所得割 × 20% ÷ (90% - 所得税率（%）) + 2,000円

2. 寄付金の妥当性
   入力した寄付合計額 vs 計算した上限額
   寄付 ≦ 上限          → ✅ 上限内（残余枠も表示）
   寄付 > 上限           → ❌ 上限超過（超過分の自己負担額を具体的に表示）

3. 控除反映の確認
   ワンストップ特例の場合：
     → 源泉徴収票の「寄附金控除の額」欄は空白が正常
     → 住民税決定通知書（翌年6〜7月）で税額控除を確認する旨を案内
     → 確定申告をした場合はワンストップ特例が無効になる点を注意喚起

   確定申告の場合：
     → 「寄附金控除の額」欄に (寄付合計 - 2,000円) が記載されているか確認
     → 確定申告書（オプション読み込み済みの場合）と突き合わせ

4. ワンストップ特例の適用可否
   自治体数 ≦ 5 かつ 確定申告不要 → ワンストップ特例が使える
   自治体数 > 5 または 確定申告あり → 確定申告で寄附金控除が必要

5. 節税効果の試算
   ふるさと納税による節税額 ≈ 寄付額 - 2,000円（自己負担）
   上限内の場合、実質的な節税額を表示
```

### 7.5 給料明細と源泉徴収票の整合チェック

```
複数月の給料明細が読み込まれた場合：

1. 月次データの集計
   総支給額の合計 → 源泉徴収票の「支払金額」と比較
   所得税の合計   → 源泉徴収票の「源泉徴収税額」と比較
   社会保険料の合計 → 源泉徴収票の「社会保険料等の金額」と比較

2. 差異の判定
   ±1,000円以内 → ✅ 一致（端数処理の範囲）
   ±1,001〜10,000円 → ⚠️ 小さな差異（賞与・交通費精算等の可能性）
   ±10,001円超  → ❌ 大きな差異（確認推奨）

3. 月次グラフ
   月別の総支給額・所得税・社会保険料の推移をバーチャートで表示
   （標準報酬月額の改定月・賞与支給月を視覚的に確認できる）
```

**給料明細との整合**
```

ℹ️ 来年（2025年）の住民税は2025年6月から変わります。
   2024年分の所得に基づき決定され、2025年6月の給与明細で確認できます。
```

## 9. エラーハンドリング

| 状況 | 対応 |
|------|------|
| PDF読み込み失敗 | エラーメッセージを表示し、手入力モードへ切り替え |
| テキスト抽出できない（スキャンPDF等） | 「このPDFは画像形式です。手動で数値を入力してください」と案内 |
| 書類種別の自動判定失敗 | ユーザーに書類の種類を選択させる |
| 抽出値が異常（マイナス・桁数不一致） | 当該項目を黄色ハイライトし「要確認」を表示 |
| 計算結果が著しく乖離 | 具体的な差異を提示し、入力値の確認を促す |
| 給料明細の月数が不足 | 「〇ヶ月分読み込み済み。12ヶ月分揃うと年間集計が行えます」と案内 |
