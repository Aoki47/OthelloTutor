# 年末調整PDF解析アプリ 仕様書

## 1. 概要

### 1.1 アプリ名（仮）
**NeNcho Check（年調チェック）**

### 1.2 目的
源泉徴収票や年末調整関連のPDFをブラウザに読み込み、以下を自動で確認・解説する：

- 所得税・住民税・社会保険料の計算が妥当か
- ふるさと納税の控除が正しく反映されているか
- 見落としがちな控除が適用されているか

### 1.3 基本方針
- **完全ブラウザ動作**：PDFを含むすべてのデータをサーバーに送信しない
- **個人利用向け**：シンプルなUI、詳細な解説テキスト付き
- **2024年度（令和6年分）の税制に対応**

---

## 2. 機能一覧

| 機能 | 概要 |
|------|------|
| PDF読み込み | 源泉徴収票PDFのテキスト抽出 |
| 自動解析 | 各項目の数値を自動認識・マッピング |
| 補足入力 | PDF非記載の情報（家族構成・ふるさと納税実績等）を手入力 |
| 税金妥当性チェック | 所得税・住民税・社会保険料の計算検証 |
| ふるさと納税確認 | 上限額・控除反映・申請方式の確認 |
| 解説レポート | 各項目に解説を付けた結果表示 |

---

## 3. 技術仕様

### 3.1 技術スタック

| 分類 | 採用技術 | 理由 |
|------|----------|------|
| フレームワーク | Vanilla JS または React（TypeScript） | 軽量・依存少 |
| PDF解析 | [PDF.js](https://mozilla.github.io/pdf.js/)（Mozilla製） | ブラウザ完結、実績あり |
| スタイル | TailwindCSS または CSS Modules | 見やすいUI |
| ビルド | Vite | 高速・設定シンプル |
| ホスティング | GitHub Pages / Vercel（静的ファイルのみ） | 無料・簡単 |

### 3.2 動作環境
- モダンブラウザ（Chrome 90+, Firefox 90+, Safari 15+）
- スマートフォンは対象外（PCブラウザのみ想定）

### 3.3 データの扱い
- アップロードされたPDFはブラウザのメモリ内のみで処理
- localStorageには補足入力値のみ任意保存（デフォルトOFF）
- 外部APIへの通信なし

## 5. PDF解析仕様

### 5.1 対象書類
- **源泉徴収票**（主対象）：会社が発行する年間給与・税額の証明書
- ※ 年末調整計算書・控除申告書は参考程度（テキスト抽出の精度依存）

### 5.2 抽出対象フィールド

| フィールド名 | 源泉徴収票上の表記 | 型 |
|------------|----------------|-----|
| 支払金額（年収） | 支払金額 | 整数（円） |
| 給与所得控除後の金額 | 給与所得控除後の金額 | 整数（円） |
| 所得控除の額の合計額 | 所得控除の額の合計額 | 整数（円） |
| 源泉徴収税額 | 源泉徴収税額 | 整数（円） |
| 社会保険料等の金額 | 社会保険料等の金額 | 整数（円） |
| 生命保険料の控除額 | 生命保険料の控除額 | 整数（円） |
| 地震保険料の控除額 | 地震保険料の控除額 | 整数（円） |
| 住宅借入金等特別控除の額 | 住宅借入金等特別控除の額 | 整数（円） |
| 寄附金控除の額 | 寄附金控除の額 | 整数（円） |
| 配偶者控除の有無・金額 | 控除対象配偶者の有無等 | bool / 整数 |
| 扶養親族の数 | 控除対象扶養親族の数 | 整数 |

### 5.3 抽出方式
1. PDF.jsでテキストレイヤーを抽出
2. 正規表現パターンで各フィールドを検索
3. 抽出に失敗した項目はUIで明示し、手入力を促す
4. 電子データ版（テキスト埋め込み）が対象。スキャンPDFは非対応（明記）

## 6. 補足入力フォーム仕様

### 6.1 家族構成
| 項目 | 入力形式 | 用途 |
|------|---------|------|
| 配偶者の有無 | チェックボックス | 配偶者控除・配偶者特別控除の判定 |
| 配偶者の年収 | 数値（円） | 配偶者特別控除の額を決定 |
| 扶養親族の数（16歳未満） | 数値 | 住民税の計算 |
| 扶養親族の数（16〜18歳） | 数値 | 控除額の計算 |
| 扶養親族の数（19〜22歳・特定） | 数値 | 特定扶養控除の適用 |
| 扶養親族の数（70歳以上）| 数値 | 老人扶養控除の適用 |

### 6.2 ふるさと納税情報
| 項目 | 入力形式 | 用途 |
|------|---------|------|
| 寄付先の自治体数 | 数値 | ワンストップ特例の可否判定（≦5箇所） |
| 寄付合計額 | 数値（円） | 控除上限・控除額の計算 |
| 申請方式 | ラジオ（ワンストップ / 確定申告 / 未申請） | 適用控除の分岐 |

| 項目 | 入力形式 | 備考 |
|------|---------|------|
| 医療費控除の申請額 | 数値（円） | 確定申告が必要な旨を注記 |
| iDeCo掛け金（年額） | 数値（円） | 小規模企業共済等掛金控除 |
| 住宅ローン借入残高 | 数値（円） | 住宅借入金等特別控除の概算確認 |

## 7. 計算・チェックロジック

### 7.1 所得税の計算検証（2024年分）

```
1. 給与所得控除の計算
   年収 → 給与所得控除額（速算表に基づく）
   給与所得 = 年収 - 給与所得控除額

2. 所得控除の積み上げ
   基礎控除：48万円（合計所得2,400万円以下の場合）
   社会保険料控除：PDF抽出値
   生命保険料控除：PDF抽出値（上限12万円）
   地震保険料控除：PDF抽出値（上限5万円）
   配偶者控除 / 配偶者特別控除（補足入力より）
   扶養控除（補足入力より）
   寄附金控除（ふるさと納税・確定申告の場合）
   住宅借入金等特別控除（税額控除として別途処理）

3. 課税所得 = 給与所得 - 所得控除合計

4. 所得税額（速算表）
   課税所得に応じた税率・控除額を適用

5. 復興特別所得税 = 所得税額 × 2.1%

6. 税額控除の適用
   住宅借入金等特別控除を差し引く

7. 検証
   計算値 vs PDF記載の源泉徴収税額
   差異が±5,000円以内 → ✅ 正常
   差異が±5,000〜50,000円 → ⚠️ 要確認
   差異が±50,000円超 → ❌ 異常の可能性
```

### 7.2 社会保険料の妥当性チェック

```
健康保険料の推定（協会けんぽの都道府県別料率で概算）
   標準報酬月額 = 年収 ÷ 12 → 等級表で決定
   健康保険料（年）= 標準報酬月額 × 料率（会社負担を除いた本人分）× 12

厚生年金保険料の推定
   標準報酬月額（上限65万円）× 9.15% × 12

雇用保険料の推定
   年収 × 0.6%（一般の事業の場合）

社会保険料合計の比較
   推定値 vs PDF記載の社会保険料等の金額
   ±10%以内 → ✅ 概ね妥当
   ±10%超  → ⚠️ 都道府県・業種等で差がある可能性を案内
```

> ※ 実際の社会保険料は都道府県・業種・標準報酬月額の等級によって異なるため、推定値との比較は目安として提示する。

### 7.3 ふるさと納税チェック

```
1. 上限額の計算（簡易版）
   住民税所得割額 = (課税所得 × 10% - 調整控除) として推定
   上限の目安 = 住民税所得割額 × 20% ÷ (90% - 所得税率) + 2,000円

2. 寄付金の妥当性
   入力した寄付合計額 vs 計算した上限額
   寄付 ≦ 上限   → ✅ 上限内
   寄付 > 上限   → ❌ 上限超過（超過分は自己負担になる旨を説明）

3. 控除反映の確認
   ワンストップ特例の場合：
     → 源泉徴収票の「寄附金控除の額」欄は空白が正常（住民税で控除）
     → 翌年6月以降の住民税決定通知で確認が必要な旨を案内
   確定申告の場合：
     → 「寄附金控除の額」欄に寄付金 - 2,000円が記載されているか確認

4. ワンストップ特例の適用可否
   自治体数 ≦ 5 かつ 確定申告不要 → ワンストップ特例が使える
   自治体数 > 5 または 確定申告あり → 確定申告が必要
```

## 8. 解析レポート表示仕様

### 8.1 ステータス表示
| アイコン | 意味 |
|---------|------|
| ✅ | 問題なし・計算が一致 |
| ⚠️ | 注意・要確認（計算誤差・推定値との乖離） |
| ❌ | 問題の可能性・確認を強く推奨 |
| ℹ️ | 解説・補足情報 |

### 8.2 表示項目と解説の例

**所得税**
```
✅ 所得税額：¥182,300（計算値 ¥181,800 との差 ¥500）
   └ 計算値との差が±5,000円以内のため、正常な範囲です。
     わずかな差異は端数処理によるものです。
```

**ふるさと納税（ワンストップ特例）**
```
⚠️ ふるさと納税：¥60,000 寄付済み（上限目安 ¥85,000）
   └ 上限まで¥25,000の余裕があります。今年まだ寄付できる枠が残っています。
     ワンストップ特例を選択中のため、源泉徴収票への記載はありません。
     住民税の決定通知書（翌年6〜7月）で控除額を確認してください。
```

**住宅ローン控除**
```
✅ 住宅借入金等特別控除：¥250,000
   └ 税額控除として所得税から直接差し引かれています。
     控除しきれない分は翌年の住民税から控除される場合があります。
```

### 8.3 サマリーカード
ページ上部にサマリーカードを表示：
- 年収
- 実質負担額（所得税 + 住民税推定 + 社会保険料）
- 実質負担率（%）
- ふるさと納税の節税効果（推定）

## 9. エラーハンドリング

| 状況 | 対応 |
|------|------|
| PDF読み込み失敗 | エラーメッセージを表示し、手入力モードへ切り替え |
| テキスト抽出できない（スキャンPDF等） | 「このPDFは画像形式です。手動で数値を入力してください」と案内 |
| 抽出値が異常（マイナス・桁数不一致） | 当該項目を黄色ハイライトし「要確認」を表示 |
| 計算結果が著しく乖離 | 具体的な差異を提示し、入力値の確認を促す |

---

## 10. 非機能要件

| 項目 | 要件 |
|------|------|
| プライバシー | PDFデータは外部送信しない。ページを閉じると消える |
| パフォーマンス | PDF読み込みから結果表示まで3秒以内（10MB以下） |
| アクセシビリティ | フォント最小16px、色だけに頼らないステータス表示 |
| 免責事項 | 「本ツールは参考情報の提供を目的としており、税務判断の根拠にはなりません」を常時表示 |
